# homebrew-monopam

Homebrew tap for OCaml tools.

## Installation

```bash
brew tap parsimoni-labs/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ofpp` | FPP-to-OCaml code generator for NASA F Prime models |

## Usage

```bash
# Install pre-built binaries
brew install ofpp

# Or build from source
brew install --HEAD ofpp
```
